The GPS Dog Collar is a project created by Becky Stern and Tyler Cooper.  For more details on the project, please visit http://learn.adafruit.com/gps-dog-collar/.

Code written by Tyler Cooper for Adafruit Industries.

To download. click the DOWNLOADS button in the top right corner, rename the uncompressed folder GPS_Dog_Collar. Check that the GPS_Dog_Collar folder contains GPS_Dog_Collar.ino.

This code example requires the TinyGPS library which can be downloaded from http://arduiniana.org/libraries/tinygps/
